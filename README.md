# 🌾 DHARTI – Smart Farming Assistant

## 🚀 Overview

**DHARTI** is an AI-powered agricultural intelligence platform designed to bridge the gap between farmers and advanced technology.

It provides:
- 🌱 Simple, actionable insights for farmers  
- 📊 Data-driven analytics for experts and institutions  

The platform follows a **dual-interface approach**:
- Farmer-friendly (easy, non-technical)
- Analyst & Institution dashboards (advanced, technical)

---

## 🧩 Features

### 🌿 Farmer Features
- 📸 Upload crop images for disease detection  
- 💬 Ask DHARTI (AI chat support)  
- 🚨 Real-time alerts (Urgent / Warning / Normal)  
- ✅ Daily task tracking  

---

### 📈 Analyst Features
- Interactive dashboards  
- Region & crop filtering  
- Disease heatmaps  
- Predictive analytics  

---

### 🏢 Institution Features
- Macro-level agricultural insights  
- Regional performance tracking  
- Data export (CSV/PDF)  
- Strategic reporting
